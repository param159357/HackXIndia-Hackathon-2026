1. PROJECT OVERVIEW
================================================================================

The Smart Traffic Violation Priority Engine is an AI-powered traffic monitoring
and emergency response system that transforms CCTV feeds into actionable
intelligence. It prioritizes traffic violations based on risk scores, with
emergency vehicles receiving infinite priority.

KEY OBJECTIVES:
- Real-time traffic violation detection
- AI-powered risk scoring (0-100 scale)
- Emergency vehicle priority override
- Interactive dashboard for traffic control centers
- Area-based violation monitoring
- Integration with Skillmap AI analysis system

TARGET USERS:
- Traffic authorities
- Emergency services
- Smart city administrators
- Law enforcement agencies

12. TROUBLESHOOTING

ISSUE: Videos not loading in grid
SOLUTION:
1. Verify assets server running on port 8002
2. Check ../skillmap/assets/ folder exists
3. Ensure video files have proper extensions

ISSUE: Map pins not appearing
SOLUTION:
1. Verify Leaflet.js loaded (check network tab)
2. Check map container height in CSS
3. Ensure click event listener attached
4. Check browser console for JavaScript errors

ISSUE: Skillmap redirect not working
SOLUTION:
1. Verify Streamlit server running on port 8501
2. Check http://localhost:8501 manually
3. Verify openSkillmapWithVideo() function
4. Check for popup blockers

ISSUE: Leaderboard not updating
SOLUTION:
1. Check areaViolations object defined
2. Verify updateAreaAlerts() called
3. Check leaderboard element exists
4. Inspect console for errors

ISSUE: Server won't start
SOLUTION:
1. Check if port already in use
2. Verify Python installed (python --version)
3. Check firewall settings
4. Try different port number
